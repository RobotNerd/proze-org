Proze Development

Proze is open source. A suite of proze-related projects are hosted on GitHub and contributions are welcome.

  • Syntax: A definition of the proze syntax, configuration options, and other related information.
  • Vim plugin: A proze plugin for the vim text editor.
  • Converter: A python application used to compile proze projects into a document for publishing.
  • The proze-org repository has the code used to generate this website.
