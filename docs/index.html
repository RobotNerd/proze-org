What is proze?

Proze is a set of open source tools that let you write English language prose the same way you would write software.

Proze is made up of these pieces:

  • The proze syntax.
  • A proze plugin for supported text editors.
  • A compiler that converts a proze project into something you can publish.

Quick start

The Getting started page provides a walkthrough of these steps.

  • Install a proze plugin in a supported editor.
  • Initialize a new proze project.
  • Create a proze config file.
  • Write your story using the *.proze format.
  • Install the proze converter.
  • Compile your proze project to one of the supported output formats.
