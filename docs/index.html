What is proze?

Proze is a set of open source tools for writing English language prose just like writing code.

Quick start

The Getting started page provides a walkthrough of these steps.

  • Install a proze plugin in a supported editor.
  • Initialize a new proze project.
  • Create a proze config file.
  • Write your story using the *.proze format.
  • Install the proze converter.
  • Compile your proze project to one of the supported output formats.
